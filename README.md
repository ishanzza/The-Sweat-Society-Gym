## Overview
This project is built with:
- **React 19**
- **TypeScript**
- **Tailwind CSS**
- **Lucide Icons**

## Features
- Modern, high-energy aesthetic ("Gymbox" inspired).
- Reveal-on-scroll animations.
- Responsive mobile-first design.
- Interactive Location map.
- Membership plan viewer and trial form.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
